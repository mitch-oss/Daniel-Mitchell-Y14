# Fixit Physio Booking System - Prototype

## Overview
A simple booking system prototype for a physiotherapy clinic. Built with Python and Tkinter.

## Files Structure
- `main.py` - Main program file (run this to start)
- `database.py` - All database operations
- `login.py` - Login screen
- `main_menu.py` - Main dashboard/menu
- `add_appointment.py` - Add new appointment form
- `view_appointments.py` - View and delete appointments

## Requirements
- Python 3.x
- tkinter (usually comes with Python)
- sqlite3 (usually comes with Python)

## How to Run
1. Make sure Python is installed on your computer
2. Open terminal/command prompt
3. Navigate to the folder containing these files
4. Run: `python main.py`
